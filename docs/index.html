Class: Token

Representa un valor, puede ser un número entero o un caracter.

Campos

Protected

int numero

Representa el valor numérico del token (en el caso que almacene uno)

char caracter

Valor de caracter (en el caso que almacene un char)

string type

Representa el tipo de valor almacenado. Valores posibles "number" o "char".

Public

Constructor

Token(int n)

Crea un token asignando el campo numero con el valor recibido. Establece el campo type en "number".

Token(char c)

Crea un token asignando el campo caracter con el valor recibido. Establece el campo type en "char".

string getType()

Retorna la cadena contenida

int getNumero

Retorna el entero asignado al token.

char getChar

Retorna el caracter asignado al token.

Class: Nodo

Representa un elemento que forma parte de una lista.

Campos

Protected

Token *dato

Almacena el token.

Nodo *next

Representa el enlace con otro nodo.

Public

Nodo()

Crea un nodo de finalización, sin dato y sin enlace.

Nodo(Token *a)

Crea un nodo asignandole como dato el parametro recibido 'a'.

void set_dato(Token *a)

Establece como dato el token recibido.

void set_next(Nodo *n)

Establece como nodo siguiente el nodo recibido.

Token *get_dato()

Retorna la referencia al token asignado como dato.

Nodo *get_next()

Retorna la referencia al nodo establecido como siguiente.

bool es_vacio()

Retorna true si el nodo siguiente es nulo y false si no lo es.
