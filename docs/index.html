Class: Token

Representa un valor, puede ser un número entero o un caracter.

Campos

Protected

int numero

Representa el valor numérico del token (en el caso que almacene uno)

char caracter

Valor de caracter (en el caso que almacene un char)

string type

Representa el tipo de valor almacenado. Valores posibles "number" o "char".

Public

Constructor

Crea un token asignando el campo numero con el valor recibido. Establece el campo type en "number".

Crea un token asignando el campo caracter con el valor recibido. Establece el campo type en "char".

string getType()

Retorna la cadena contenida

Retorna el entero asignado al token.

Class: Calculadora

Clase encargada de resolver una cadena de texto con operaciones matemáticas básicas.

Campos

Protected

bool ok

Contiene un booleano que indica si la cadena ingresada es correcta o no.

Pila *pilaOriginal

Referencia a la Pila donde se almacenan los valores de la cadena a resolver.

bool isnum(char valor)

Retorna true si el caracter recibido es un número y false si no lo es.

bool parentesis(string cad)

Retorna true si los parentesis de la expresión recibida son correctos y false si hay algún problema.

bool logica(string cad)

Retorna true si la expresion tiene sentido en términos de orden de caracteres o si no lo es.

bool analizar(string cad)

Retorna true si la cadena es correcta y se puede procesar dentro de una Pila, si no es correcta devuelve false.

Pila *invertirPila(Pila *original)

Retorna una Pila que es la inversa de la Pila recibida.

void operar(Pila *p, int a)

Procesa el entero recibido en la Pila recibida.

int calcular(Pila *p)

Procesa y resuelve operando sobre la pila recibida.

Public

Calculadora(string cad)

Crea una calculadora y analiza la cadena recibida.

void tryCalcular(Pila *p)

Si la cadena ingresada está en condiciones de ser operada, realiza las operaciones y devuelve el resultado. Caso contrario imprime un mensaje de error.
